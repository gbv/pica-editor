PicaEditor

PicaEditor is a web component to inspect and edit PICA+ records. It is based on Vue.js 3 and CodeMirror. Features include:

  • syntax highlighting
  • load records via unAPI
  • quick keyboard navigation (hit ↹ Tab to jump through subfields)
  • validation and documentation (based on Avram schemas):
    • document field and subfields at the cursor
    • autocomplete (hit ↹ Tab after the subfield indicator $) for define fields and subfields
    • detect and mark unknown fields and subfields

Feature requests and bug reports are welcome (in English or German)! Background information about PICA is available in the German handbook on PICA data.
